# 🧪 Interactive Periodic Table & Learning Suite

An interactive, responsive, and data-driven chemistry application built with modern JavaScript. This project serves as both a reference tool and an educational game for students and chemistry enthusiasts.

## 🚀 Live Demo
You can access the live version of the project here:  
**[https://ahmetyayik.github.io/Interactive-Periodic-Table/](https://ahmetyayik.github.io/Interactive-Periodic-Table/)**

## ✨ Key Features
* **Real-time Data Fetching:** Utilizes an external API to retrieve up-to-date information for all 118 elements.
* **Smart Search:** Instant filtering system by element name or symbol.
* **Chemical Group Filtering:** One-click highlighting for Noble Gases, Alkali Metals, Halogens, and more.
* **Favorites System:** Persistent storage using `localStorage` to save and track specific elements across browser sessions.
* **Interactive Quiz Module:** A built-in knowledge game with randomized questions and pedagogical feedback.
* **Fully Responsive:** Optimized for desktop, tablet, and mobile viewing with a horizontal scrollable periodic grid.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Grid Layout)
* **Framework:** Bootstrap 5 (UI Components & Modals)
* **Language:** Vanilla JavaScript (ES6+)
* **Data Handling:** Fetch API & JSON Processing
* **Storage:** Web Storage API (localStorage)

## 📖 Learning Objectives
This project focuses on:
1. **Asynchronous JS:** Managing API calls with `.then()` patterns.
2. **DOM Manipulation:** Dynamically generating 118 element cards.
3. **State Management:** Synchronizing data between Table, Search, and Quiz modules.
4. **UX/UI Design:** Implementing active states and responsive layouts.
